# Burning Night 

畢業專題最後正式名稱：Burning Night  
預告片連結：[觀看預告片](https://youtu.be/RZSAlYSrwRk?si=sYPP82SLPman-fMR)

##  遊戲簡介
在蔽日至暗之夜，點亮希望之火用心中殘存的希望，焚燒黑夜。
是一款3D快節奏動作遊戲。
## 主要機制
###  **敵人具備快速連招與高傷害輸出**
